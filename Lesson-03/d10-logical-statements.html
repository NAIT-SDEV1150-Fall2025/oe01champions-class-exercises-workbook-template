Logical Statements

We will be dealing with logical statements. Logical statements allow us to make multiple paths in our code. Depending on the outcome of a certain expression, we will follow one code path or another.

IF-ELSE Statements

There are different logic statements, and We will start with if and if else statements.

We can make decisions in our code using if and if else statements. It is very much like this template:

if *some condition is true*, Then *a certain action will happen*, else *another action will happen*

ELSE block is optional. Why? Because, expression given in brackets of IF statement will be first executed and translated to a BOOLEAN i.e. either TRUE or FALSE.

If, BOOLEAN is FALSE, then Else block will be executed.

if(expression)

{

code associated with the "IF block" is executed and if the expression is true, control will exit the complete IF/ELSE block.

}

else

{

this code will only be executed if the expression is false.

}

SWITCH Statements

If else statements are great for evaluating Boolean conditions. There are many things you can do with them, but in some cases, it is better to replace them with a switch statement. This is especially the case when evaluating more than four or five values.

Syntax:

switch(expression) {
case value1:
//statement to execute
break;
case value2:
//statement to execute
break;
}
