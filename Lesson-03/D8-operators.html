There are different types of JavaScript operators:

Aritmatic Operators

  • Addition Operator - Used to add Values or STRINGS
  • Subtraction
  • Multiplication
  • Exponentiation
  • Division
  • Modulus Division
  • Increment operator (++)
  • Decrement operator (--)

Assignment Operators (=)

Comparison Operators (= = =), (>), (<), (>=), (<=), (!=)

Logical Operators - AND (&&), OR(||) and NOT(!(X>Y))

Operand, are the subjects to the operators. So, if we say x + y, x and y are operands.

Unary operators: If we only need one operand for the operators

Binary operators: Two operands

Ternary operators: Three operands

Increment and Decrement operators are known as unary operators. x++ or ++x
